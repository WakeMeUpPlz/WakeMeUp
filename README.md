# 프로젝스 소개 #
기존의 알람앱 기능에, 패턴풀기 횟수 초과로 실패시 지정한 도우미에게 문자를 전송하는 알람 어플리케이션 
* 사용한 언어/툴 
  * front-end : kotlin/android studio
  * back-end : js, node-js/visual studio code, aws
* 사용 라이브러리 
  * 패턴풀기 : https://github.com/l7naive/pattern-lock.git
  * 클라이언트-서버 통신 : Retrofit2
